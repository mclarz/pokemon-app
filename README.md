# pokemon-app


1. ``git clone <REPO_URL>``
2. ``git checkout master``
3. ``composer install``
4. ``cp env-example .env``
5. ``php artisan migrate``
6. ``php artisan db:seed``
7. ``npm install``
8. ``php artisan serve``



Just create account first and login, go to this page http://127.0.0.0:8000/home#
